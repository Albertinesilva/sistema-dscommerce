# 🛒 DSCommerce - E-commerce Backend

Backend application for an e-commerce system developed with Spring Boot, focusing on domain modeling, data persistence and layered architecture.

---

## 🚀 Objective

This project aims to demonstrate the implementation of a backend system for an e-commerce application, covering essential concepts such as domain modeling, entity relationships and data persistence using JPA.

---

## 🛠️ Technologies

- Java
- Spring Boot
- Spring Data JPA
- Hibernate
- H2 Database
- Maven

---

## 📚 Concepts Covered

- 🧱 Domain modeling
- 🔗 Entity relationships
- 🛒 E-commerce business rules
- 🗄️ Data persistence with JPA
- 🏗️ Layered architecture
- 🔄 CRUD operations

---

## 🧩 Domain Overview

The system models a basic e-commerce structure, including:

- 👤 Users (clients and admins)
- 📦 Products
- 🧾 Orders
- 📁 Categories
- 💳 Payments

---

## 📂 Project Structure

- `entities/` → domain model
- `repositories/` → data access layer
- `services/` → business logic
- `controllers/` → API layer

---

## ▶️ Running the project

1. Clone the repository:

```bash
git clone https://github.com/Albertinesilva/sistema-dscommerce
```

2. Navigate to the folder:

```bash
cd sistema-dscommerce
```

3. Run the application:

```bash
./mvnw spring-boot:run
```
Or run it via your IDE.
